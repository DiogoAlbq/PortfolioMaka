# MAKA | Commissions & Portfolio

Este é o site de portfólio e comissões do **MAKA**, com design moderno, responsivo, seguro e multilíngue (Português/Inglês).

## 🚀 Tecnologias Utilizadas

- **React** (v19)
- **Vite** (v6)
- **Tailwind CSS** (v4)
- **Motion** (para animações fluidas)
- **Lucide React** (para ícones)

## 📦 Como Rodar Localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse no navegador em: `http://localhost:5173`

## 🛠️ Build de Produção

Para gerar a build estática otimizada para o GitHub Pages:
```bash
npm run build
```
Os arquivos prontos para deploy serão gerados na pasta `dist/`.

## 🔒 Segurança & Performance

- **Segurança**: Implementação de política CSP (Content Security Policy) estrita e cabeçalhos de política de referência/permissões.
- **Acessibilidade**: Elementos semânticos HTML5 e tags ARIA integrados para melhor legibilidade por leitores de tela.
- **SEO**: Meta tags Open Graph configuradas para compartilhamento otimizado em plataformas como Discord, Twitter, etc.
